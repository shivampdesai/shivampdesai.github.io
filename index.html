Projects

Gunrock

Gunrock is a CUDA library for graph-processing designed specifically for the GPU. It uses a high-level, bulk-synchronous, data-centric abstraction focused on operations on a vertex or edge frontier. Gunrock is featured on NVIDIA's list of GPU Accelerated Libraries as the only external library for GPU graph analytics.

BillSplit

I built the BillSplit iOS application to address one of the most frequent problems I faced in my college years. Splitting a bill amongst a group of friends was always more complicated than it needed to be, especially when it came to groceries with my roommates. The application calculates each individual's totals based on the items they are actually going to use, instead of splitting it evenly.
